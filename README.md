# AKINORIV
## Сайт виронички [akinoriv.com](http://akinoriv.com)

### Инструкция по запуску на сервере
#### Первый запуск

```shell
cd ~ &&
git clone https://github.com/Akinoriv/web-akinoriv.git && 
cd web-akinoriv && 
chmod +x startup.sh && 
sudo ./startup.sh
```

#### Запуск после обновления

```shell
cd ~/web-akinoriv && 
git pull && 
sudo ./startup.sh    
```
